• INTRODUCTION TO STATISTICS
This web application is developed for introducing people to basic statistical terms with the help of Scalable Vector Graphics(SVG), JavaScript, and interactive animations. The platform explains basic concepts in statistics in interactive manner so that learners can engage with the content while learning the concept by 'feeling' it.

As of 2021-06-30, it includes the following topics:

  • Mean ---> Go to mean page
  • Median, Percentile, Quartile, Histogram ---> Go to median page
  • Variance and Standard Deviation ---> Go to variance and standard deviation page

In near feature, the following topics will be included:

  • Correlation
  • Types of variables
